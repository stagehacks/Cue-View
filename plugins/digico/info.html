Cue View emulates to be the DiGiCo SD iPad app.
  1. On the SD console open Master Screen → Setup → External Control
  2. Enable External Control by pressing the button at the top of the panel
  3. Press the add device button and select DiGiCo Pad
  4. Enter a Device Name for Cue View and then enter the IP Address of this computer
  5. Enter Send and Receive Port numbers
    • Cue View Port = DiGiCo Receive
    • Cue View Local = DiGiCo Send
  6. Tick the Enable column for this device
  7. Press the Load button in the bottom right corner of the panel and select the commands button for the relevant console
