Connection Requirements: Eos 3.2 and newer

Browser: Setup → Device Settings → Network

  • Enable Third Party OSC on the network interface(s) you are using
  • Ensure this Cue View device's port is set to 3037

 

Connection Requirements: Eos 3.1 and earlier

Shell/ECU → Network

  • Enable ✔ TCP OSC. The TCP format should be "TCP Format for OSC 1.1 (SLIP)".
  • Enable Third Party OSC
If the "Third Party OSC" option is not available, Eos must be updated to 3.1 or newer.

Browser: System Settings → Show Control → OSC

  • Enable ✔ OSC RX
  • Enable ✔ OSC TX
  • All other fields may be left to defaults or blank.
  • If a custom port is desired specify it in under OSC TCP Server Ports.
